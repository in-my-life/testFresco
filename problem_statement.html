Welcome to Git: Merging a branch - Basic (Java).

This question is used to evaluate your skills on merging a branch in Git.

For this, you will be evaluated based on the following operations. The remote repository has been preconfigured into your local git repository.

  1. Your local git repository will be linkedlist_java.
  2. There are 2 branches namely master and new_branch. Both of which are in sync with the remote repository.
  3. Merge the new_branch branch into master branch.
  4. Push the updated master branch to remote repository.

Passing marks: 100%

Note 1: To get familiarized with the Hacker Rank environment, please watch the tour video.

Note 2: The branch names must not be misspelled.

Note 3: The IDE's Git will not work in the local repository. Use Terminal for it.
